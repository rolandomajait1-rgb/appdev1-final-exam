# React Todo App - APPDEV1 Final Exam

A complete React Todo application built with Redux Toolkit, React Router, and JSONPlaceholder API integration.

## Live Demo
You can view the live version of the app here:  
**https://rolandomajait1-rgb.github.io/appdev1-final-exam/**

## Features
- ✅ User Authentication with JSONPlaceholder API
- ✅ CRUD Operations for Todos
- ✅ Theme Switching (Standard, Light, Darker)
- ✅ Real-time Clock Display
- ✅ Redux State Management
- ✅ Responsive Design
- ✅ React Router Navigation

## Technologies Used
- React 19
- Redux Toolkit
- React Router DOM
- Axios for API calls
- Vite for build tooling
- GitHub Pages for deployment

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Deploy
```bash
npm run deploy
```
